# 📚 Library Management System (Qt & C++)

## 📌 Project Overview
This project is a **GUI-based Library Management System** developed using **C++** and **Qt Framework** as part of an **Object-Oriented Programming (OOP)** group project.  
The system is designed to automate and manage common library operations such as book management, member management, and issue/return tracking through a user-friendly graphical interface.

- Apply **Object-Oriented Programming concepts** in a real-world application

- Improve teamwork, version control, and software design skills
